# Daily Habits Board API

An API designed to track personal habits and their progress. Examples of habits include “drink water,” “read for 20 minutes,” or “stretch your back.” Each habit stores a frequency and a counter for completed days.

## Features

- Register new habits  
- Mark a habit as completed  
- View simple progress statistics  
- Filter habits by category  
- Practice clean models, validations, and RESTful routing

## Purpose

This project provides a simple environment to strengthen your understanding of REST APIs in ASP.NET Core. It focuses on clear route design, data validation, and lightweight CRUD operations.

## Requirements

- .NET SDK  
- Any editor (VS Code recommended)
